Food Production "Make Tofu, Soymilk, and Tempeh from Soybeans and Vinegar"

1. Description :

  • Boil Soybeans, blend, separate okara and ferment with Rhizopus oligosporus starter. Add vinegar to some soymilk as coagulant, sweetening the remainder.
