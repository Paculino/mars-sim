Process : "Reduce Gypsum To Calcium Carbonate With Carbon"

1. Description :

  • Wrapping two processes: CaSO4 * 2H2O (gypsum) + 2C -> CaS + 2CO2 + 2H2O CaS + H2O + CO2 -> CaCO3 (calcium carbonate) + H2S Note: ignoring hydrogen sulfide as waste for now. Gypsum has up to 8.6 % water


2. Characteristics :
Required Building Tech Level    1
Required Skill Level 1
Work Time in millisols 100.0
Time in millisols 200.0
Power Requirement 0.1

  		• Resource     7.0     Carbon

  		• Resource     12.7     Carbon Dioxide
