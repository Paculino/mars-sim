What's New in v3.4.0

14 Jun 2022

A. New Features

  1. Airlock : Implement airlock reservation system for EVA egress.
  2. Base Mass : Add base mass to all buildings, in preparation for calculating overall rocket payload mass.
  3. Building : Add building general info such as dimensions and mass.
  4. Building Window: New dedicated Building Unit Window which replaces the Building tab in the Settlement window.
  5. Computing : Add new Computation building function, new Computer Science subject, new Computer Scientist job, new Computing specialist and Chief of Computing role, and new Computing skill.
  6. Filter : New Settlement filter and Building tab to the Monitor Tool.
  7. Inventory : Replace with lightweight inventory for all units. Revamp the storage of resources/equipment and transfer of person/vehicle between locales.
  8. Mission : Restructure and simplify handling of Mission.
  9. Parts : Add garment.
  10. Person : Make a person's carrying capacity to be age-dependent.
  11. Trading : Compute and relate the cost, profit, and price of a good.
  12. Scenario : Arriving future Settlements are configured from a scenario. Scenarios can be exported and shared.
  13. Social : Remove global relationship graph and simplify codes.
  14. Waste : Split grey/black water & solid waste management resource processes into a separate waste processing function.
  15. Scenario : Edit and load basic scenarios.
  16. Authority Editor : Customize sponsors/authorities.
  17. Location : Show areocentric longitude of Mars (where Mars is at in the solar system) in Time Tool.
  18. Economic : Display the cost/profit/price of a good in each settlement for comparison.

B. Improvements

  1. Make a person's carrying capacity age-dependent.
  2. Enable lightweight inventory for all units.
  3. Significantly reduce CPU usage by implementing process interval in resource processing, air composition and crop farming.
  4. May bypass the text console menu and go straight to starting in Sandbox Mode.

C. Fixes

  1. Consider loading/Unloading edge cases for Trading/Delivery.
  2. Resolve stalled delivery negotiation.
  3. Exploration mission revisits existing sites until they reach an evaluation ready for Mining.
  4. Ground vehicle maintenance no longer stalls when EVA is aborted.
  5. Fix the mission selection that stop Navigation Tab from displaying sites and navpoints.
  6. Correct vehicle fuel calculation.
