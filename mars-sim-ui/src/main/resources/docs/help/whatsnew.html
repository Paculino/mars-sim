3.5.0 (build 7906) - 3 Apr 2023

CORE ENGINE IMPROVEMENTS :

  1. Battery : Standardize battery capacity.
  2. Events : Supports future scheduled events to better handle periodic actions on the entities.
  3. Mission : Add deadline on rover mission departure. Split embarking phase into loading & departing.
  4. Parts : Add silica aerogel as transparent rooftop tiles.
  5. Robot : Define specifications using RobotSpec class.
  6. Shift : Add shift manager to handle work shift change and add on leave status.
  7. Stock Cap : Increase general/cargo capacities and specific resource storage capacity.
  8. Storage : Allow selection of storage bin in DigLocal task based on building resource capacity.
  9. Task : Define SettlementMetaTask and track a pool of shared tasks.

UI IMPROVEMENT :

  1. Desktop : Remember tools' position and contents.
  2. L&F : Look and Feel can be changed via Settings menu item.
  3. Mission : Organize missions under a tree structure for each settlement.
  4. Panels : Separate tables for Maintenance, Waste and Resource processing.
  5. Styling : Single styling via look and feel applied to all components.
  6. Tab : Add new tab to the Monitor Tool to show task backlog of each settlement.

RUNTIME ENVIRONMENT :

  1. Java : Require Java 17 or above.

FIXES :

  1. EVA Suit : Allow suit repair in a vehicle using repair task.
  2. Jar file : Make settlement template name readable by using lowercase letters.
  3. Robot : Correct robot battery status to be based on percentage, instead of actual KWh.
  4. Vehicle : Correct fuel consumption calculation.
