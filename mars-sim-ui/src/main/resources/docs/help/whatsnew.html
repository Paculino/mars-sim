What's New in v3.4.0

26 May 2022

A. New Features

  1. Implement airlock reservation system for EVA egress.
  2. Add new Computation building function, new Computer Science subject, new Computer Scientist job, new Computing specialist and Chief of Computing role, and new Computing skill.
  3. Add base mass to all buildings, in preparation for calculating overall rocket payload mass.
  4. Add building general info such as dimensions and mass.
  5. Arriving future Settlements are configured from a scenario.
  6. Scenarios can be exported and shared.
  7. Add garment.
  8. New Settlement filter and Building tab to the Monitor Tool.
  9. New dedicated Building Unit Window which replaces the Building tab in the Settlement window.
  10. Show areocentric longitude in Time Tool.

B. Improvements

  1. Make a person's carrying capacity age-dependent.
  2. Enable lightweight inventory for all units.
  3. Significantly reduce CPU usage by implementing process interval in resource processing, air composition and crop farming.
  4. May bypass the text console menu and go straight to starting in Sandbox Mode.

C. Fixes

  1. Consider loading/Unloading edge cases for Trading/Delivery.
  2. Resolve stalled delivery negotiation.
  3. Exploration mission revisits existing sites until they reach an evaluation ready for Mining.
  4. Ground vehicle maintenance no longer stalls when EVA is aborted.
