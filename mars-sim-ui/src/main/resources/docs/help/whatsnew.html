What's New in v3.4.0

27 Jun 2022

CORE ENGINE IMPROVEMENTS :

  1. Airlock : Implement airlock reservation system for EVA egress.
  2. Base Mass : Add base mass to all buildings, in preparation for calculating overall rocket payload mass.
  3. Building : Add building general info such as dimensions and mass.
  4. Computing : Add Computation function, Computer Science subject, Computer Scientist job, Computing specialist and Chief of Computing role, and Computing skill.
  5. Filter : New Settlement filter and Building tab to the Monitor Tool.
  6. Inventory : Replace with lightweight inventory for all units. Revamp the storage of resources/equipment and transfer of person/vehicle between locales.
  7. Mission : Restructure and simplify handling of Mission.
  8. Parts : Add wearing garment.
  9. Person : Make a person's carrying capacity to be age-dependent.
  10. Trading : Compute and relate the cost, profit, and price of a good.
  11. Scenario : Arriving future Settlements are configured from a scenario. Scenarios can be exported and shared.
  12. Social : Remove global relationship graph and simplify codes.
  13. Vehicle : Rework vehicle specifications. Refine fuel economy calculation.
  14. Waste : Split grey/black water & solid waste management resource processes into a separate waste processing function.

UI IMPROVEMENT :

  1. Authority : Add editor that customize sponsors/authorities.
  2. Economic : Display the cost/profit/price of a good in each settlement for comparison.
  3. Location : Show areocentric longitude of Mars (where Mars is at in the solar system) in Time Tool.
  4. Scenario : Edit and load basic scenarios.
  5. Window : Add newly dedicated Building Unit Window which replaces the Building tab in the Settlement window.

FIXES :

  1. (Un)loading : Consider loading/unloading edge cases for Trading/Delivery.
  2. Delivery : Resolve stalled negotiation.
  3. Exploration : revisits existing sites until they reach an evaluation ready for Mining.
  4. Maintenance : Ground vehicle maintenance no longer stalls when EVA is aborted.
  5. Mission UI : Fix the mission selection that stop Navigation Tab from displaying sites and navpoints.
  6. Vehicle : Correct vehicle fuel calculation.
