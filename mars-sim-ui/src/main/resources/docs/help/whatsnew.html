What's New in v3.4.0

09 Nov 2022

CORE ENGINE IMPROVEMENTS :

  1. Base Mass : Add base mass to all buildings, in preparation for calculating overall rocket payload mass.
  2. Building : Add building general info such as dimensions and mass.
  3. Computing : Add computing function and skill. Track computing resource usage.
  4. Filter : New Settlement filter and Building tab to the Monitor Tool.
  5. Inventory : Replace with lightweight inventory for all units. Revamp the storage of resources/equipment and transfer of person/vehicle between locales.
  6. Mission : Restructure and simplify handling of Mission.
  7. Parts : Add wearing garment.
  8. Person : Track exercise. Make a person's carrying capacity to be age-dependent.
  9. Processing : Unify resource and waste processes, food production and manufacturing under common super class.
  10. Robot : Track battery capacity and charging.
  11. Trading : Compute and relate the cost, profit, and price of trade goods.
  12. Scenario : Arriving future Settlements are configured from a scenario. Scenarios can be exported and shared.
  13. Site : Explore, claim and track mining sites.
  14. Social : Remove global relationship graph and simplify codes.
  15. Sunlight : Predict sunrise and sunset at each settlement. Track actual sunrise and sunset time.
  16. Meta Task : Add TaskJob and remove duplicate computation of outstanding tasks.
  17. Vehicle : Rework vehicle specifications. Refine fuel economy calculation.

UI IMPROVEMENT :

  1. Authority : Add editor that customize sponsors/authorities.
  2. Economic : Display the cost/profit/price of a good in each settlement for comparison.
  3. Icons : Replace word title of each tab with an icon in all Unit Windows.
  4. Location : Show areocentric longitude of Mars in Time Tool.
  5. Power : Display generated/load power & stored energy for settlements in Monitor Tool.
  6. Scenario : Edit and load basic scenarios.
  7. Window : Add new Building Window with tab, replacing old Building tab.

FIXES :

  1. Airlock : Fix bugs during EVA egress and ingress.
  2. Delivery : Resolve stalled negotiation.
  3. Exploration : Revisit existing sites until they reach an evaluation ready for Mining.
  4. Maintenance : Ground vehicle maintenance no longer stalls when EVA is aborted.
  5. Mission : Fix the mission selection. Correct Navigation tab.
  6. Settlement : Correct settlement selection in Monitor Tool.
  7. Trading : Consider loading/unloading edge cases for Trading/Delivery.
  8. Transport : Correct handling of arriving settlement by transport manager.
  9. Vehicle : Correct vehicle fuel calculation.
