What's New

Version pre-3.6.0 - 2 Jun 2023

A. CORE ENGINE IMPROVEMENTS :

  1. Crop : Add a new category herbs
  2. Earth Time : Model using the standard Java LocalDateTime class
  3. Elevation : Adopt higher resolution height data
  4. Fuel : Enable the use of methanol as fuel for vehicles
  5. Power : Model fission reactor with more parameters
  6. Resupplies : Define resupply missions with a repeating schedule
  7. Vehicle : Enable regen braking and encapsulate propulsion calculations in motor controller class
  8. Vehicle : Maintains own reference to assigned Mission.
  9. Mission : Retains reference to Vehicle after Mission completed.

B. UI IMPROVEMENT :

  1. Resupply : Showcase resupply missions by settlements under a tree.
  2. Role : Restore previous role if change is not confirmed.
  3. Mars Map : Use a variety of resolutions of surface maps.
  4. Events : Events in the Monitor Window are clickable to drill down into the details of the entity.
