Resource : "Bassanite" (Solid)


1. Description :

  • Bassanite is a calcium sulfate mineral with formula CaSO4·0.5 or 2CaSO4·H2O. In other words it has half a water per CaSO4 unit hence its synonym hemihydrate. A regolith derived mineral deposit
